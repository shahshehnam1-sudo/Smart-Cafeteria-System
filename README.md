# Smart Cafeteria Ordering System

An OOP-based cafeteria ordering system built as a semester project.

# Features
- Browse menu items
- Add/remove items from order
- Quantity control per item
- Auto bill calculation
# OOP Concepts Used
- Classes & Objects
- Encapsulation
- Inheritance
- Polymorphism
